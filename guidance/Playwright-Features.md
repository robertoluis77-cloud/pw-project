# Playwright Features

A list of key Playwright features with links to the official documentation:

- **End-to-End Test Framework**  
  Bundles test runner, assertions, isolation, parallelization, and rich tooling.  
  [Docs: Introduction](https://playwright.dev/docs/intro)

- **Cross-Browser Support**  
  Test on Chromium, WebKit, and Firefox across Windows, Linux, and macOS.  
  [Docs: Browsers](https://playwright.dev/docs/browsers)

- **Native Mobile Emulation**  
  Emulate Chrome (Android) and Mobile Safari.  
  [Docs: Mobile Emulation](https://playwright.dev/docs/mobile)

- **Headless and Headed Modes**  
  Run tests in headless or headed mode.  
  [Docs: Running Tests](https://playwright.dev/docs/running-tests)

- **Parallel Test Execution**  
  Run tests in parallel for speed.  
  [Docs: Test Parallelism](https://playwright.dev/docs/test-parallel)

- **Rich Test Reports**  
  Generate HTML reports for test results.  
  [Docs: Test Reporters](https://playwright.dev/docs/test-reporters)

- **UI Mode for Debugging**  
  Watch mode, live step view, and time travel debugging.  
  [Docs: UI Mode](https://playwright.dev/docs/test-ui-mode)

- **Codegen for Test Generation**  
  Generate tests by recording user actions.  
  [Docs: Codegen](https://playwright.dev/docs/codegen-intro)

- **Centralized Configuration**  
  Configure browsers, timeouts, retries, projects, and reporters.  
  [Docs: Test Configuration](https://playwright.dev/docs/test-configuration)

- **Web-First Assertions and Locators**  
  Write robust tests using web-first assertions and locators.  
  [Docs: Writing Tests](https://playwright.dev/docs/writing-tests)

For more, visit the [Playwright documentation](https://playwright.dev/docs/intro).
